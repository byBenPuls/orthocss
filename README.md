# OrthoCSS

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

OrthoCSS is a **clean, minimalist, and systematic CSS framework** for styling all standard HTML elements elegantly and consistently.  
It is inspired by the [Tacit framework](https://github.com/yegor256/tacit) and aims to give developers:

- Minimal CSS without classes
- Light, Dark, and Auto theme support
- JavaScript-free styling
- Consistent design for headings, lists, tables, forms, code, and more

## Installation / Usage


There are **three themes** you can use:


### 1. Auto Theme
Automatically switches based on the user's system preference:

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/byBenPuls/orthocss@v1.0.0/dist/auto.css">
```

### 2. Light Theme
For a bright/light interface:

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/byBenPuls/orthocss@v1.0.0/dist/light.css">
```

### 3. Dark Theme
For a dark interface:

```
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/byBenPuls/orthocss@v1.0.0/dist/dark.css">
```
