Introduction

OrthoCSS is a minimalist CSS framework designed to style all standard HTML elements elegantly and consistently. Its name comes from the Greek word orthos — meaning correct, straight, precise. The framework is inspired by the Tacit framework by Egor Bugaenko.

OrthoCSS supports both light and dark themes, making it versatile for different design needs. It is JavaScript-free and encourages avoiding CSS classes entirely: you can create fully styled pages using only standard HTML elements. This approach removes unnecessary complexity and lets you focus on content and readability.

“Precision in design is the foundation of readability and elegance.”

How to get started

There are three simple ways to use OrthoCSS:

  1. Auto theme: 
    <link rel="stylesheet" href="https://orthocss.dev/orthocss-auto.css">
  2. Light theme: 
    <link rel="stylesheet" href="https://orthocss.dev/orthocss-light.css">
  3. Dark theme: 
    <link rel="stylesheet" href="https://orthocss.dev/orthocss-dark.css">
